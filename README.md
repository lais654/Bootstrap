Loja Tech é uma página web desenvolvida em HTML utilizando o Bootstrap para criar uma interface moderna e responsiva. O objetivo é apresentar uma vitrine virtual com produtos de tecnologia de forma organizada e visualmente
agradável.

Funcionalidades:
Cabeçalho com nome da loja e botão de destaque.
Exibição de três produtos em formato de cards.
Cada produto possui imagem, descrição e botão de compra.
Rodapé com informações da loja.
Tecnologias utilizadas.
Estrutura da página:
Header: nome da loja e apresentação.
Seção de Produtos: exibição dos produtos em cards.
Footer: mensagem de direitos reservados.

O projeto demonstra como criar uma página simples de uma loja virtual, utilizando recursos do Bootstrap para facilitar o desenvolvimento e melhorar a aparência da interface.
